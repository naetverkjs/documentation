Naetverk

Open Source Web Framework for visual programming

Get Started

Modular

With a Event-based architecture, it is easy to add new functionality in to a existing process.

Flexible

You can extend the existing functionality by creating new node and component types, connectors or sockets.

Independant

Naetverk is not bound to a frontend framework. The base library can be used in any Framework that handles the rendering.
